Ce jeu d'échec est fonctionel. Il faut ouvrir le fichier display.py et le lancer pour voir apparaître l'échéquier. 
Pour bouger une pièce, il faut la sélectionner en cliquant dessus. 
Ensuite, des cases en jaune s'affichent. Ce sont les cases sur lesquelles la pièce choisie peut se déplacer. 
Il suffit alors de cliquer sur la case sur laquelle on souhaite déplacer la pièce.
